# Brave Adventurers Guild

Um jogo de gerenciamento e extração side-scroller vertical para Windows, focado em acessibilidade profissional e lógica baseada no SRD d20.

## Tecnologias
- **Engine:** Godot Engine (C#)
- **Acessibilidade:** Tolk DLL para integração com NVDA/JAWS.
- **Sistema:** Baseado no SRD d20 (Pathfinder/D&D 3.5).

## Estrutura
- `Scripts/Logic`: Core do jogo e regras d20.
- `Scripts/Access`: Integração com leitores de tela.
- `Assets/DLLs`: Dependências nativas (Tolk.dll).
